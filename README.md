# Portfolio

This repository contains the source and assets for my professional portfolio website. It showcases a curated selection of projects, concise case studies, and supporting materials that demonstrate my technical skills, problem-solving approach, and the outcomes I delivered.

## About

I am a software developer focused on building reliable, maintainable web applications and interactive experiences. My work combines front-end development, practical UI/UX considerations, and solid engineering practices to deliver business value.

## Highlights
- Selected projects with implementation details and live demos or repository links.
- Short case studies describing goals, challenges, and results.
- A summary of core technologies and tools used.

## Technologies
Common technologies used across projects: HTML, CSS (Bootstrap), JavaScript, and related tooling.

## How to view
- Open `index.html` in a browser.
- Or serve the folder locally (recommended) and visit `http://localhost:8000`:

```bash
# Node (serve):
npx serve .

# Python 3:
python -m http.server 8000
```
